MAC MAMP环境PHP添加pcntl扩展

pcntl 扩展可以支持 PHP 的多线程操作。（非Unix类系统不支持此模块）

首先将mac上的默认php修改为MAMP中的php版本也就是不使用mac自带的步骤如下：

  1. 打开terminal终端 sudo vim ~/.bash_profile 打开或新建该文件后在文件末尾添加如下信息

  2. 保存 执行source ~/.bash_profile后php -v查看php版本是否切换成功

下面演示的是给 MAMP的PHP.6.32版本添加 pcntl扩展。

官网下载和本地 mamp中PHP版本对应的源码包，php官网源码

1. 解压源码包

2. 编译 pcntl.so 文件

在编译pcntl.so之前文件网上还有说添加以下零时环境变量（本机安装测试时添加以下环境变量编译报错无法正常生成pcntl.so扩展）

3. 向php.ini文件中添加该扩展 重启apache
